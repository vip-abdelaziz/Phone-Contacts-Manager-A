#  Phone Contacts Management System
An advanced C++ application developed using Object-Oriented Programming (OOP) principles. 
This project demonstrates efficient data management and clean code architecture.

##  Technical Implementation
* **Modular Design:** Separation of concerns using Header (`.h`) and Source (`.cpp`) files.
* **Encapsulation:** Secure data handling using Private members and Public interfaces.
* **Dynamic Memory:** Efficient use of `std::vector` for real-time data storage.
* **Formatted UI:** Clean CLI (Command Line Interface) for better user experience.

##  How to Run
1. Clone the repository.
2. Compile `main.cpp` using any C++ compiler (GCC/Clang/MSVC).
3. Ensure `Contact.h` is in the same directory.
